# Description : 
Ce projet est une série de travaux pratiques en JavaScript avancé (ES6+) visant à approfondir les concepts clés du langage.
Il aborde les closures, les fonctions avancées et la programmation fonctionnelle avec map, filter et reduce.
La gestion du contexte this, l’utilisation de call, apply et bind sont également mises en pratique.
Les étudiants manipulent les prototypes et les classes ES6, incluant l’héritage.
La programmation asynchrone est traitée via les Promises, async/await et les combinateurs de promesses.
Un accent particulier est mis sur la gestion des erreurs, les délais et les mécanismes de retry.
Les TP avancés exploitent le DOM, l’event delegation et la persistance avec LocalStorage.
Les données sont manipulées à partir de structures JSON.
Le projet final consiste en un dashboard étudiant interactif avec recherche, filtre et tri.
L’objectif global est de produire un code fonctionnel, propre et conforme aux bonnes pratiques.
